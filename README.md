# Study Chill Rooms

Web tao phong hoc nho, co chat, binh chon, boc tham, bat/tat mic/camera va chia se man hinh.

## Chay local

```bash
npm start
```

Mac dinh web chay tai:

```text
http://localhost:5501
```

## Dua len online cho moi nguoi dung

Web nay can `server.js` de tao phong, chat va gui tin hieu, nen khong nen dua len GitHub Pages don thuan. GitHub Pages chi phu hop web tinh HTML/CSS/JS, khong chay duoc Node server.

Cach de dung duoc cho moi nguoi:

1. Tao repo moi tren GitHub.
2. Upload tat ca file trong thu muc nay len repo.
3. Vao Render.com, tao `New Web Service`.
4. Ket noi voi repo GitHub vua tao.
5. Chon:
   - Build Command: `npm install`
   - Start Command: `npm start`
6. Deploy xong, Render se cap cho ban mot link cong khai dang `https://ten-web.onrender.com`.

Neu Render doc duoc `render.yaml`, cac cau hinh tren se duoc dien san.
